# ChatTI - Chatbot de TI em Python com Tkinter

## Descrição
ChatTI é uma aplicação desktop desenvolvida em Python usando Tkinter que simula um chatbot especializado em assuntos de TI. O aplicativo possui login, interface gráfica profissional com suporte a modo claro e escuro, exibição de ícones e nomes nas mensagens, e respostas automáticas baseadas em palavras-chave de tecnologia.  

## Funcionalidades
- Tela de login com validação de usuário e senha.
- Tela principal de chat com:
  - Área de mensagens com scroll.
  - Campo de entrada de texto.
  - Botões "Enviar", "Limpar" e "Alternar Tema".
- Exibição do nome e ícone do usuário e do chatbot em cada mensagem.
- Mensagens formatadas com cores, alinhamento e espaçamento distintos.
- Respostas automáticas sobre temas de TI (Python, Docker, Git, SQL, Linux, Cloud, Front-end, APIs, Deploy, etc.).
- Simulação de "digitando..." antes da resposta do bot.
- Suporte a modo claro e escuro.
